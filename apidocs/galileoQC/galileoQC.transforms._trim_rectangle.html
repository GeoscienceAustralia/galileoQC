galileoQC.transforms._trim_rectangle¶

Trim a grid along each edge.

Author: Mark Helm Dransfield

Created: Oct 2025

Module Contents¶

Functions¶

_trim_rectangle

Returns a smaller grid trimmed along each edge. Simply find the smallest and largest easting and northing that have a non-Nan value, and trim to the rectangle so defined.

API¶

galileoQC.transforms._trim_rectangle._trim_rectangle(grid)¶

Returns a smaller grid trimmed along each edge. Simply find the smallest and largest easting and northing that have a non-Nan value, and trim to the rectangle so defined.
