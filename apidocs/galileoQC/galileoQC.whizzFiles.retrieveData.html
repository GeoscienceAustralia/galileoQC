galileoQC.whizzFiles.retrieveData.getLineXChannel(whizzFile, line, x, channel)¶

Returns 1D numpy arrays of x and channel in line from geoWhizz filename. The inputs are just two channels in the data and the names ‘x’ and ‘channel’ do not carry intrinsic meaning.

Intended use for analysis and plotting.

Parameters

whizzFile : String or pathlib Path

Name of a HDF5 Whizz file, including path and extension.

line : String

A flightline, e.g. '1000110.0'.

x : String

The name of the x variable.

channel : String

The name of the channel.

Returns

xData : numpy 1D array

A numpy array of data, float32 or float64.

yData : numpy 1D array

A numpy array of data, float32 or float64.
