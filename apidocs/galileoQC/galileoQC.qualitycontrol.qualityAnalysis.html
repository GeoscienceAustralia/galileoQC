checkVertAcc

Uses numpy.diff to estimate the vertical acceleration from the vertical velocity data (units of m/s) in the channel called vertvelocity in whizzFile. Plots the result (units of milli-g) for each survey line.

Uses numpy.diff to estimate the vertical acceleration from the vertical velocity data (units of m/s) in the channel called vertvelocity in whizzFile. Plots the min, max, mean and stdev (units of milli-g) for each survey line as a function of line. Assumes samples are 1 sec apart.

Uses numpy.diff to estimate the vertical acceleration from the vertical velocity data (units of m/s) in the channel called vertvelocity in whizzFile. Plots the result (units of milli-g) for each survey line.

Parameters

whizzFile : HDF5 Whizz file pathlib Path

The pathlib Path to the Whizz HDF5 file containing the survey line data.

lines : Array{String}, optional

Array of line numbers as strings. Default = [], meaning all lines are checked.

None

galileoQC.qualitycontrol.qualityAnalysis.checkStatcor(whizzFile, statcor, flight='', lines=[])¶

Plots statcor vs flight as a scatter plot. Used to compare with static gravimeter readings.

Parameters

whizzFile : HDF5 Whizz file pathlib Path

The pathlib Path to the Whizz HDF5 file containing the survey line data.

statcor : String

The name of the channel containing the static corrections.

flight : String, optional

The name of the channel containing the flight number. The default is to use the line attribute.

lines : Array{String}, optional

Array of line numbers as strings. Default = [], meaning all lines are checked.

Returns

None

galileoQC.qualitycontrol.qualityAnalysis.lineStats(whizzFile, channel, lines=[])¶

Generate a statistical plot for the channel across all lines. The plot shows the min, mean, max and stdev for the channel as a function of line number.

Parameters

whizzFile : String or pathlib Path

Name of a HDF5 Whizz file, including path and extension.

channel : String

The name of the channel to plot.

lines : Array{String}, optional

Array of line numbers as strings. Default = [], meaning all lines are checked.

Returns

None.

Plot the statistics of channel1 - channel2 in each flightLine.
