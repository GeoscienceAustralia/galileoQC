galileoQC.whizzFiles.colReader.colsToHDF(col_datfile, lineColumn, dateColumn, latColumn, lonColumn, dumstr='*', whizzFile='')¶

Reads the data from the ASCII survey file and writes it to a new Whizz HDF5 survey file.

Parameters

col_datfile : pathlib.PurePath The pathlib Path to the input ASCII data file. whizzFile : pathlib.PurePath, optional The pathlib Path to the output Whizz HDF5 file. The default is ‘’ and the output file is then the same as the input file with the extension changed to ‘.hdf5’. lineColumn : Integer The column number containing the line numbers.

Returns

whizzFile : pathlib.PurePath The name of the created whizz file.
