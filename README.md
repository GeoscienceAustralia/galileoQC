# AirGravQC
A python package to quality check the airborne gravity dataset

**Package Installation and Setup on National Computing infrastructure**

	a. Setting up the python environment in NCI

Using the instructions in this link, and set up a python3 environment in your specific location. For memory/space limitations for each user, it is suggested that you make this custom python environment in your project directory. The environment will have a folder with the same name in your working directory after it is built:

For the QAQC python scripts, we need to install these extra python3 packages within our new python environment:

	-Xarray (version 2022.12.0 or higher)
	-Rioxarray
