# Airborne Gravity Quality Control

**AirGravQC** is a Python library for quality assurance and control of airborne gravity data. It covers both gravimetry and gradiometry from a range of sensor models and was designed for use in QC of data collected under contracts based on the Geoscience Australia Airborne Gravity Deed , although it is useful for the QC of most airborne gravity projects.

__note__

   This project is under active development.
